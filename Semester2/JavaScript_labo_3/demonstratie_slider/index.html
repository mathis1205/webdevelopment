Waar wordt de event listener gekoppeld aan de slider?

In de setup function.

Waarom moeten we die op twee soorten events koppelen?

Als je enkel de change koppelt zal hij enkel het eindresultaat in de console plaatsen.
Als je enkel de input koppelt geeft hij elke waarde waar je over gaat bij het sliden.

In de CSS file wordt nergens een rode kleur opgegeven, waar wordt dan wel de rode kleur van het blokje ingesteld?

In de setup function in de JavaScript file.

Waarom schrijven we telkens sliders[0] en colorDemos[0] en niet gewoon sliders of colorDemos?

Omdat er telkens een NodeList wordt weergegeven als je getElementsByClassName gebruikt.
Je moet dus telkens het eerste (en ook het enige) element uit die lijst halen.
