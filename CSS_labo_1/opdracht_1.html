Wat zijn de ancestors van het tweede <li> element?

  • De ancestors van het tweede <li> element in deze dom tree zijn:
    1. Het <ul> element
    2. Het <body> element
    3. Het <html> element

Wat zijn de descendants van het <body> element?

  • De descendants van het <body> element in deze dom tree zijn:
    1. Het <h1> element
    2. Het <p> element
    3. Het <ul> element
    4. De <li> elementen

Hoeveel elementen in een geldig HTML document hebben geen parent?

  • Enkel het HTML element

Heeft elk element één of meerdere child elementen?

  • Neen, de elementen die helemaal onderaan de dom tree staan hebben geen child elementen.
